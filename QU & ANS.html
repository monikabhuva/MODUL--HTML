Q6. what is the 'class' attribute in HTML? with Example

Ans.

The 'class' attribute specifies one or more classname to the element.
The 'class' attribute is mostly used to point to a class in a style sheet. However it can
also be used by a javascript to make changes to html elements with a specified class .

Example

< p id="iddemo" > Demo for id selector < /p >
< style >
#iddemo{
color: green;
}
< /style >

Example of class attribute:

< h2 class="header" > Hello < /h2 >
< style >
.header{
color:blue;
}
< /style >

What is difference between HTML and XHTML?

Ans.

HTML XHTML
HTML stands for Hypertext Markup Language. XHTML stands for Extensible Hypertext Markup Language.
HTML is not case sensitive. XHTML is case sensitive.
HTML is not mandatory for a single root element. XHTML documents must contain at least one root element.
All content can be included inthe body element. All content must be put in block.
Attribute values are not significant in HTML. Attribute values are important in XHHTML.
There is no hard rule on the structure of the elements. The of the elements should be followed.
HTML is less expressive. XHTML is more expressive as compared to HTML.
